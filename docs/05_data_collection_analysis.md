Titel
Datensammlung und Analyse
Datensammlung
Nutzer sprechen mit dem Prototyp. Die Gespraeche werden gespeichert.
Die Daten enthalten Fragen, Antworten und Zeitpunkte.
Analyse
Die Daten werden mit Stimmungsanalyse und Themenanalyse untersucht.
Die Analyse sucht nach
positiven Aussagen
negativen Aussagen
konkreten Reaktionen auf Hinweise
Ausgabe
Die Resultate werden in Abbildungen und kurzen Beschreibungen festgehalten.
Der Code und die Analyseprompts liegen im Ordner analysis.
