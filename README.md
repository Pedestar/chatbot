Conversational Intervention Projekt

Inhalt
Dieses Projekt untersucht, wie ein digitales Dialogsystem das Wohlbefinden junger Erwachsener beeinflusst.
Die Arbeit umfasst Motivation, Zielgruppe, messbare Wirkung, Prompts, Prototyp, Datensammlung und Analyse.

Ordner
docs beschreibt Motivation, Wirkung, Prompts, Prototyp und Analyse
data enthaelt Gespraeche aus Tests
analysis enthaelt Auswertungscode und Ergebnisse
prompts enthaelt Prompts als Textdateien
bot enthaelt Dateien aus dem verwendeten Framework
